# Printing instructions

**If you are using Duramic PLA+, you can follow these print instructions. If you are using a different plastic, the density will be different. Check weight from the slicer estimate before printing.**

Each part will be printed with either "solid" or "regular" print settings.

### "Solid"
- 2 walls
- Alternate extra wall (OrcaSlicer will change another setting when this is enabled)
- 100% infill
- Concentric infill pattern
- Internal solid infill line thickness double that of walls. (If using a .4 nozzle, this setting should be .8)

These settings are from a [Makers Muse](https://www.youtube.com/watch?v=c7CI6yBTKMc) video on Youtube.

### "Regular"
- 4 walls
- 15% infill
- Supports enabled
- Support critical regions only
- Supports from bed only

## Parts with each setting
All Duramic PLA+ parts should be printed with a nozzle temperature of 240 C.

The shark arm is white, the rest of the parts are blue.

Solid parts:
- Shark arm
- Forks (right and left)

Regular parts:
- All other parts (except the zip tie)

Print the zip tie in PETG with 99 walls.
